subsurf-queueclient

Queue client for subsurf, using the SRC API (see Github).Please note:

  • Runs can be sorted and/or filtered, see below.
  • 20'000 is the maximum of runs that can be loaded.
  • Runs will be cached on the SRC API and might be already verified or rejected. It can take up to 30 minutes to get new data from the API.
  • You can sort for duplicates, but also filter for them (filter for the term "duplicate"). Only the duplicates will have the attribute, not the original submissions.
